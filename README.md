# A.I.M. Knowledge Base

Philosophy, benchmarks, design history, essays, and behind-the-scenes thinking for Actual Intelligent Memory (A.I.M.).

### How to browse this repository
This repository is designed as a **public Obsidian vault**.

The best way to read this documentation is to:
1. Clone or download this repository.
2. Open the folder directly in the [Obsidian app](https://obsidian.md/) using "Open folder as vault".
3. Explore the interconnected knowledge graph, backlinks, and tags natively.
