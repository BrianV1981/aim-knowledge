# A.I.M. Knowledge Base & Public Vault

Welcome to the **A.I.M. Knowledge Base**. This repository holds the deep philosophy, raw benchmark logs, architectural design history, manifestos, and behind-the-scenes thinking for the [Actual Intelligent Memory (A.I.M.)](https://github.com/BrianV1981/aim) project.

## Why does this repository exist?
A.I.M. uses a **Decoupled Exoskeleton** architecture. To prevent "content shock" for new users, the main A.I.M. repository's wiki is ruthlessly pruned down to only the absolute essential onboarding steps (Quickstarts and Installation Guides).

All of the deep "lab notes," raw JSON session logs, and long-form essays outlining the convergent evolution of AI memory are offloaded here into this dedicated digital garden. 

## How to browse this knowledge base

There are two ways to explore this repository:

### 1. The GitHub Wiki (Quick Reading)
You can read all the documents directly on GitHub using the built-in Wiki interface:
👉 **[Read the A.I.M. Knowledge Base on GitHub Wiki](https://github.com/BrianV1981/aim-wiki/wiki)**

### 2. As a Public Obsidian Vault (Interactive Graph)
This repository is natively designed as a **Public Obsidian Vault**. All documents use native `[[wiki-style]]` links. To get the full experience and see the interconnected knowledge graph:
1. Clone or download this repository to your computer.
2. Open the folder directly in the [Obsidian app](https://obsidian.md/) using "Open folder as vault".
3. Explore the knowledge graph, backlinks, and tags natively.
